# homebrew-trod

Homebrew tap for [trod](https://github.com/yestheboxer/trod) — persistent directory history with interactive TUI picker.

## Install

```
brew install yestheboxer/trod/trod
```

## Usage

After installing, add to your shell config:

```zsh
# ~/.zshrc
eval "$(trod init zsh)"
```

```bash
# ~/.bashrc
eval "$(trod init bash)"
```
